# Lendsqr Wallet Service

A backend wallet service for Demo Credit mobile lending app.

## Features
- User registration with blacklist check
- Wallet funding
- Transfer between users
- Withdrawal functionality
- Transaction history

## Tech Stack
- Node.js (LTS)
- TypeScript
- Express.js
- MySQL with Knex.js ORM
- Jest for testing

## Setup Instructions
1. Clone repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure
4. Start MySQL: `docker-compose up -d`
5. Run migrations: `npm run migrate:latest`
6. Start server: `npm run dev`
